# covid19tracker
Covid 19 Tracker build in Vue js 3 and Tailwind
API from "www.covid19api.com"

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
